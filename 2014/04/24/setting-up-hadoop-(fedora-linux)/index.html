Setting up Hadoop (fedora-linux)

Choose your way of installation

There are actually two ways of installing Hadoop:

  • Download source code and compile it;
  • Download distribution and use it. The problem of the 2nd approach is that the pre-compiled distribution is a 32-bit version, and it generates a warning every time a command is executed when running on a 64-bit machine. But generally you would get into much more troubles if you try to compile from the source code. So here we use the 2nd approach and just ignore the warnings. The distribution can be downloaded from hadoop's website. In case you want to access API docs in an IDE such as eclipse, you can also download the source code and attach it to appropriate packages.

Hadoop in Eclipse

There are several ways of integrating Hadoop into Eclipse. We again choose the easiest one: hadoop eclipse plugin. You can build from the source or use the release version if you are working on Hadoop 2.2.0 and Eclipse Kepler. Then you can simply create a new Map/Reduce Project in Eclipse, add a Map/Reduce Location, and run examples such as WordCount. By default, the input and output files locate on local filesystem.
